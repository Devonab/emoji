It's an open-source project which permits you to copy easily an emoji (it's an emoji picker, basically).
Μπορείτε να αναζητήσετε με λέξεις-κλειδιά ή να χρησιμοποιήσετε το "τα πιο χρησιμοποιούμενα emojis". Ή απλά ψάχνει για το τέλειο emoji πλοήγηση στις ενότητες.

https://github.com/Sav22999/emoji

Μπορείτε να προσαρμόσετε το addon, στην πραγματικότητα μπορείτε να ενεργοποιήσετε ή να απενεργοποιήσετε διάφορες επιλογές, οι οποίες καθιστούν το πρόσθετο προσωπικό και μοναδικό!

Μπορείτε να χρησιμοποιήσετε τη συντόμευση: Ctrl+Alt+A στα Windows και Linux και Cmd+Alt+A στο macOS.

--

Μερικές πληροφορίες σχετικά με τις γραμματοσειρές που χρησιμοποιούνται στην εφαρμογή:
- Η κύρια γραμματοσειρά και, κατά τη γνώμη μου, το πιο όμορφο είναι το Twemoji, που αναπτύχθηκε από το Twitter. Μπορείτε να διαβάσετε περισσότερα για αυτό δωρεάν (είναι επίσης open-source!) εδώ: https://twemoji.twitter.com/

- OpenMoji, η οποία είναι η δεύτερη αγαπημένη μου emoji-γραμματοσειρά. Διαβάστε περισσότερα εδώ: https://openmoji.org/

- Noto sans, by Google. Διαβάστε περισσότερα εδώ: https://www.google.com/get/noto/help/emoji/

Άλλες γραμματοσειρές, όπως το "OS emoji font" εξαρτάται από τις ρυθμίσεις OS σας, έτσι θα πρέπει να τις δείτε άδειες.
